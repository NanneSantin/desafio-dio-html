Estrutura Básica do Documento HTML

A estrutura básica do documento HTML fornece um esqueleto para organizar o conteúdo e as configurações de uma página web.

  1. <!DOCTYPE html>: define o tipo de documento HTML que está sendo usado. Nesse caso, indica que o documeto HTML está sendo escrito em conformidade com os padrões da especificação HTML5
  2. <html>: é o elemento raiz do documento HTML. Todos os outros elementos HTML são filhos diretos ou indiretos dessa tag. Ela envolve todo o conteúdo da página.
  3. <head>: é usada para incluir metadados e informações sobre o documentos HTML, como título, links para folhas de estilo CSS, JavaScript, metatags e outras configurações. O conteúdo inserido dentro dessa tag não é exibido na página, mas fornece informações para o navegador e mecanismos de busca.
  4. <body>: é o local onde o conteúdo visível da página é inserido. Ela contém todos os elementos HTML que serão exibidos no navegador, como texto, imagens, links, listas, formulários, etc.
Veja abaixo como fica a estrutura básica do documento HTML: 
<!DOCTYPE html>
<html>
    <head>
    </head>
    <body>
    </body>
</html>

Tags Básicas

Antes de apresentar algumas tags básicas, vamos ver o que são tags pair (ou tags contêiner) e tags self-closing (ou tags vazias):

Tags pair ou Tags contêiner

São usadas para envolver e delimitar o conteúdo que desejamos aplicar uma formatação ou estrutura específica. A estrutura básica desse tipo de tag é <html> </html>, onde a primeira é a tag de abertura e a segunda é a tag de fechamento. O conteúdo fica entre elas e os atributos ficam dentro da tag de abertura.

Tags self-closing ou Tags vazias

São usadas para indicar a presença de um elemento ou conteúdo adicional ou sem a necessidade de uma tag de fechamento. Sendo assim, ela só possui a tag de abertura.

Algumas tags

  • <p> </p>: usada para fazer parágrafos;
  • <blockquote> </blockquote>: usada para criar um bloco de citação;
  • <strong> </strong>: usada para deixar o texto em negrito;
  • <u> </u>: usada para deixar o texto sublinhado;
  • <i> </i>: usada para deixar o texto em itálico;
  • <mark> </mark>: usada para deixar o texto marcado;
  • <sup> </sup>: usada para criar texto sobrescrito, ou seja, texto que é exibido acima da linha de base normal exemplo;
  • <hr>: cria uma linha horizontal;
  • <br>: insere uma quebra de linha;
  • <small> </small>: deixa o texto pequeno;
  • <font> </font>: era amplamente usada no passado para definir formatação de texto, como cor, tamanho e família da fonte. No entando, ela foi considerada obsoleta no HTML5 e é recomendado usar CSS para estilizar o texto.
  • <del> </del>: é usada para indicar a exclusão ou remoção de texto em um documento html, exemplo;
  • <abbr> </abbr>: é usada para definir uma abreviação ou sigla em um documento HTML. Pode ser exibida em tooltip quando o usuário passa o mouse sobre ela, dependendo do estilo aplicado.

Atributos

Os atributos fornecem informações adicionais sobre elementos (tags). Eles são usados para modificar o comportamento ou a apareência dos elementos, fornecer metadados ou associar informações específicas a eles. São especificados dentro da tag de abertura de um elemento e possuem um nome e um valor. O nome do atributo é seguido pelo símbolo de igual (=) e o valor é definido entre as aspas duplas ("") ou simples ('').

Vamos ver um exemplo:

<img src="imagem.png" alt="Imagem de exemplo" height="100" width="100">

No exemplo usamos a tag de inserção de imagem, e temos os seguintes atributos:

  • src: atributo que informa onde a imagem está localizada (pasta local ou URL);
  • alt: texto alternativo que aparece quando a imagem não carrega por algum motivo;
  • height: altura em pixels da imagem;
  • width: largura em pixels da imagem;

Atributos genéricos

  • id: é um atributo usado para atribuir um identificador exclusivo a um elemento, permitindo selecioná-lo de forma precisa e aplicar estilos, manipulação de elementos ou interações específicas com base nesse identificador através de CSS ou JavaScript.
  • style: é um atributo usado para definir estilos de formatação, como cores, fontes e tamanhos, diretamente dentro do elemento ou da página.
  • class: é um atributo usado para atribuir uma classe ao elemento, permitindo a aplicação de estilos ou manipulação específica através de CSS ou JavaScript.

Atributos específicos (exemplos)

  • type: é uma atributo específico da tag <input>;
  • src: é um atributo específico da tag <img>;
  • href: é um atributo específico da tag <a>;

Listas

Lista Ordenada

Para criar uma lista ordenada segue-se a estrutura abaixo:

<ol>
    <li> Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 </li>
</ol>

Sendo que:

<ol> </ol>

identifica uma lista ordenada.

Irá aparecer assim:

  1. Item 1
  2. Item 2
  3. Item 3

Lista Não Ordenada

Para criar uma lista não ordenada segue-se a estrutura abaixo:

<ul>
    <li> Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 </li>
</ul>

Sendo que:

<ul> </ul>

identifica uma lista não ordenada.

Irá aparecer assim:

  • Item 1
  • Item 2
  • Item 3

A tag <li> </li> é a responsável em criar os itens das listas.

Links

Os links servem para criar ancoragem na página e / ou redirecionar para outras páginas.

A tag utilizada para criar os links é <a> </a> e a sua estrutura básica é: 

<a href="https://www.dio.me/"> Dio.Me </a>

Fica assim: Dio.me e ao clicar nesse link irá te direcionar para a página da DIO.

Alguns atributos dessa tag:

  • href: é o atributo onde inserimos a URL ou o ID de onde queremos ancorar (quando queremos direcionar para outro local na mesma página).
    • Para realizar a ancoragem colocamos href="#nomedoid", ou seja, "#" + o nome do ID que você definiu em algum lugar da sua página que você quer direcionar o usuário ao clicar no link.
  • target: é o atributo onde podemos definir se ao clicar no link o mesmo será aberto na mesma janela (target="_self") ou em uma nova janela (target="_blank").
  • title: é o atributo que cria aquele balão que aparece quando passamos o mouse em cima do link.
